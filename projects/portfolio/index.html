about me

as a person who loves logic and problem solving, i've studied maths acquiring a msc in the process (pdes & convex analysis). nevertheless, i've always been fascinated by computers and technology; spending a bunch of my free time learning new programming concepts and writing  —  debugging :'(  —  code.

among other things, i'm interested in web developing; combining html, css and js to create beautiful, dynamic and memorable designs.

in the future, i'd love to learn more about data science and contribute to some open source chess projects.
